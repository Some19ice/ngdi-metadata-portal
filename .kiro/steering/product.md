# NGDI Metadata Portal

The National Geospatial Data Infrastructure (NGDI) Metadata Portal is a comprehensive web-based platform for Nigeria's geospatial data infrastructure. It serves as a centralized hub enabling government agencies, organizations, and the public to discover, access, and contribute to the nation's geospatial data resources.

## Core Features

- **Metadata Management**: Create, edit, and manage metadata records following international standards
- **Advanced Search**: Text, spatial, and faceted search capabilities for geospatial data discovery
- **User & Organization Management**: Comprehensive role-based access control with node officers
- **Admin Dashboard**: System monitoring, user management, and configuration tools
- **Interactive Map Viewer**: Visualize and interact with geospatial data using Leaflet and MapLibre
- **GIS Services Integration**: Support for ArcGIS REST API, OGC WMS/WFS services
- **API Access**: Programmatic access to metadata and data services

## Target Users

- Government agencies managing geospatial data
- Organizations contributing to Nigeria's spatial data infrastructure
- Node officers responsible for organizational data management
- System administrators overseeing the platform
- Public users discovering and accessing geospatial resources

## Key Workflows

- Metadata record creation and validation
- Spatial and text-based data discovery
- User role and permission management
- Organization-based data governance
- GIS service integration and validation
